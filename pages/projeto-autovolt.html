The Challenge

Baterias Moura produces thousands of units daily. The legacy system suffered from siloed data and 24-hour latency, making real-time defect detection impossible. We needed a pipeline that could ingest, clean, and serve insights in under 30 seconds.

THE ARCHITECTURE

01
raw_on

Bronze Layer

Raw ingestion from IoT sensors and SAP APIs. Immutable storage of historical data in Avro format to preserve schema evolution.

02
cleaning_services

Silver Layer

Cleaning, filtering, and de-duplication. Data is transformed into structured Delta formats, enabling time-travel and ACID compliance.

03
insights

Gold Layer

Business-level aggregates. Optimized for Looker and PowerBI consumption. Real-time dashboards for floor managers and executives.

TRANSFORMATION_LOGIC

The core transformation uses BigQuery SQL and dbt (Data Build Tool) to orchestrate the transition from Bronze to Silver. We implemented custom logic to handle late-arriving telemetry data from the production line.

  • check_circle Incremental loading reduces compute cost by 45%
  • check_circle Automated data quality checks using Great Expectations
AUTOVOLT_PIPELINE.SQL 
WITH raw_telemetry AS (
  SELECT
    sensor_id,
    timestamp_trunc(event_time, SECOND) as sec_ts,
    avg(voltage) OVER (PARTITION BY sensor_id) as avg_v
  FROM `moura-production.bronze.iot_stream`
)
INSERT INTO `moura-production.silver.production_metrics`
SELECT * FROM raw_telemetry
WHERE avg_v > 12.6 -- Filter Outliers
